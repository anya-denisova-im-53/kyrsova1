# kyrsova1
Конструктор "Your BTS Playlist Creator" Проєкт, що імітує функціонал Spotify.
**Your BTS Playlist Creator** — це музичний вебсайт (імітація стрімінгової платформи на зразок Spotify), створений спеціально для шанувальників відомого K-pop гурту BTS.

💜💜Сайт допомагає користувачам створювати власні музичні добірки, додавати улюблені треки (наприклад, *Dynamite, Butter, Fake Love*) до черги відтворення та налаштовувати їхній порядок за допомогою гнучкої системи пріоритетів.💜💜
